CSS 禁止文字选中

发表于 2017-01-06 | 更新于 2018-04-16 |
# CSS

现象


html中可能有些地方不想让用户复制文字，或是用a标签做了个点击按钮，点快的时候文字会被选中。

解决方案：

不同的浏览器设置的内容不一样，user-select不是一个W3C的标准，浏览器的支持不完整，需要对不同的浏览器进行调整。
user-select有2个值（none表示不能选中文本，text表示可以选择文本）
